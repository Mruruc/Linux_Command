# `mkdir` Command

## 📁 Basic Usage

- **`mkdir directory_name`**:  
  Create a single directory (folder).

## 📂 Multiple Directories

- **`mkdir dir1 dir2 dir3 etc.`**:  
  Create multiple directories at once.

## 🌳 Nested Directories

- **`mkdir -p parent_directory/child_directory/grandChild_directory`**:  
  Create nested directories in a single command.

## 📝 Verbose Mode

- **`mkdir -v directory_name`**:  
  Display a message for each created directory.

## 🔒 Permissions

- **`mkdir -m MODE`**:  
  Create a directory and specify the rwxr-xr-x permissions.
