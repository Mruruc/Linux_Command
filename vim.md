## ✒️ `vim` Editor

- **`vim file_name.extension`**:  
  Open the Vim editor.

### 📝 Modes:

- **`i`**:  
  Enter insert mode.

### 🔐 Exiting Vim:

- **`esc` + `:`**  
 
  Enter command mode.

  - Without saving changes: **`:q!`**
  - Save changes and exit: **`:wq`**

### ⏪ Actions:

- **`u`**:  
  Undo the last action.

- **`Ctrl + r`**:  
  Redo the previous undo.

- **`:w`**:  
  Save changes without exiting.
