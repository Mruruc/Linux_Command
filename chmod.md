# üîê `chmod` Command

## **Types of Permissions:**
- **`r`**: Read
- **`w`**: Write
- **`x`**: Execute

## **Sets of Permissions:**
- **User (`u`)**:  
  The owner of the file or directory.
  
- **Group (`g`)**:  
  Users who are members of the file's group.
  
- **Others (`o`)**:  
  All other users.

## File and Directory Indicators:
The first character of the permission string indicates the type:
- **`-`**: Regular files
- **`d`**: Directories
- **`l`**: Symbolic links
